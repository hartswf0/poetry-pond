Poetry Pond · Shared Index

A text that remembers what the reader did. Every build below is one attempt at the same contract.
The rubric chain (Coaxing the Ripples · Marttila · Hartsoe · Bolter): action → stored change → later output shifts → reader can detect it. Miss one link and the ripple fails. If the berry returns untouched, the system forgot.
